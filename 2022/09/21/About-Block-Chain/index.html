About Block Chain

Posted on 2022-09-21 In Web3 , BlockChain

從 Alpha Camp 畢業後，順利申請上了 Z institute 學院的 Web3 前端開發課程交換學生，稍微記錄一下開學至今近兩週的學習近況。

AlphaCamp 畢業後成為的交換 Z institute 學生

從 Alpha Camp Web2.0 的全端網頁開發課程成功畢業，到順利申請到交換學生去開始學區塊鍊技術，對於一個幾乎是新手小白的我來說，常常聽到區塊鏈、加密貨幣、NFT…，但這些東西對我來說都好陌生，完全跟 Web2.0 所學的東西，只掛上一點點邊，也剛好趁這個難得的機會可以透過整個課程，有人整理完善的資訊，跟著學校一起學習這塊未來的趨勢。

從頭學習

因為對這方面的知識幾乎說是 0，所以在開課前的一週，學校開出了開課前的預習作業，必須自行去搜尋資料去了解很多相關的知識，每週要自修閱讀的東西真的多到爆炸，每週幾乎要花上 20 個小時以上去做延伸閱讀跟自學，雖然忙跟累可是卻感覺很充實，收穫滿滿，並且期待課程結束後，能夠和組員一起完成一個屬於 Web3 概念的去中心化網頁應用程式專案。（給自己加油，握拳 ✊)

初嘗試申請錢包地址

在 Web3 的世界裡，完全是使用加密貨幣在做交易以及執行很多相關的程序，所以加密錢包是絕對必要的，所有的 Web3 Web App 基本上都需要連結加密錢包。
加密錢包的申請方式，需要先安裝加密錢包(也可稱之為 Provider，有桌機或手機錢包應用)，在透共錢包應用來申請，申請時會提供加密的私鑰、或是助記詞來幫助你找到你的錢包在網路上的位置，所以要非常小心被駭，或是遺失這些私鑰或助記詞，一旦遺失或落入別人手中，你將會遺失你所有的虛擬資產，所以在資安管理上要非常小心。

而我第一次申請錢包是透過 TokenPocket 來申請錢包，TokenPocekt 在申請時會給予 12 個英文單字做為助記詞，而助記詞透過不可逆的加密後則會產生私鑰，所以這兩者都不能遺失，只要遺失你將失去所有虛擬資產。

第二次申請錢包是透過現在很熱門的 MetaMask 錢包來申請，但 MetaMask 錢包僅會提供私鑰，並不透過助記詞來加密產生私鑰，雖然私鑰本身已經是透過加密的一串亂碼，但有助記詞是更為方便記憶，因為一長串的亂碼是很難背或記得。

助記詞與私鑰的保存

關於私鑰或助記詞，絕對別給別人，通常也建議使用紙筆來記錄，或存在一個需要指紋辨識或是生物特徵辨識的儲存裝置，如果不擔心 USB 遺失，也可以放在 USB 裡，但是 USB 也有一天壞的可能，絕對別存放在電腦裡或是存在雲端或是以照片方式儲存，這些都是有可能被駭客取得的方式，也別隨意點奇怪的連結被釣魚，或安裝奇怪來路不明的外掛程式。

EPI 與 ERC-20、ERC-721、ERCf-1155

簡單說明 EPI 就是以太坊的改進提議，每個 EIP 都會有編號，作為 Web3 世界開創的前幾大佬，這個 ERC-20 加密貨幣就是由乙太坊的 Frozeman 所提出連結在此

任何人都可以在 GitHub 上的 Ethereum 官方帳號裡找到這些議題
而這些 Issues 都會有編號，而 ERC-20 代幣就是在這個 Issues 裡面編號為 20 的議題，ERC-20 加密貨幣的就是從此議題討論最後產生出來的。

ERC-20 也稱為同值性代幣，所有的加密貨幣都可以彼此轉換，但因為加密貨幣又跟現實金錢有匯率關係，所以也不見得是 1:1 兌換，只是他們是同性質，可以依照虛擬貨幣的價值做不同虛擬貨幣幣的兌換，而在以太坊最大的交易所及就數 Uniswarp。

但作為有些虛擬產物，例如數位藝術作品、音樂、遊戲道具、認證證明…諸多虛擬產物，他們並不具同價值性，所以才產生了 ERC-721 非同值性代幣(也就是 NFT，Non-Fungible Token，不可取代的 Token)，這些 ERC-721 產物擁有獨一無二的代碼，也會記錄產生的日期，以及自身編號與總數量，並且所有的 NFT 是無法直接互相兌換，必須透過 ERC-20 虛擬貨幣來交易，但也並非所有 ERC-721 產物都可以轉移，這要看當初在鑄造此產物時的設定，例如有些認證證書就不該可以轉移，必須綁定在個人錢包上，亦或是由某些非去中心化組織來管理。

而後來因為 ERC-721 只能作為單筆交易使用，但遊戲道具或數位收藏品有可能有成千上百萬種，有時候交易可能不只一項物品，所以衍伸出了 ERC-1155，ERC-1155 協議就是能夠透過單筆交易多項物品，此後也更加便利於虛擬世界上的交易。

初嘗試購買加密貨幣

透過各式各樣的加密錢包應用，可以連接到相關的中間平台商，能夠使用信用卡來購買這些虛擬的加密貨幣，而虛擬貨幣的匯率是浮動，所以在購買時也有很多相關需要注意的事情。(以後有機會再補充)

初購買 NFT

最近嘗試有在 Opeansea 跟 X2Y2 上嘗試購買 NFT，Opensea 為一個 NFT 交易平台，也可以透過此網站來鑄造 NFT 產品，而最近就在這上面購買了此大頭照

所以目前就可以透過查看錢包地址來知道此圖片的過往來源，來確認此圖片的所有權是我的 🤣，而在未來可能相關的社交軟體都可以將所有的個人照片或是圖片上傳至區塊鍊中，給予這些圖片獨一無二無法複製的 Token，就可以追查這些圖片產權屬於誰，同時也代表了更難去冒充假冒別人，因為太容易查詢到圖片跟帳號的歸屬是誰了。

熱門公鏈與 Web3 網頁應用

透過學校的介紹，得知這個叫 Dapp 從這網站可以取得所有公鏈平台上的 Dapp（去中心化網頁應用程式），相當好用，可以依照不同的鏈或是 Dapp 性質來過濾及熱門度來排序。

結語

目前區塊鍊應用技術正在快速的蓬勃發展，用途上其實也非常多的可能性，不僅僅只是加密貨幣而已，雖然暫時還不普及，但這就像是一個大混亂時代的過渡期，就像從以前我們只能閱讀網站無法跟網站互動，只能讀取網站給我們的資料。（Web1.0，只能讀）

到現在能夠在網站上，下單購買商品，留言貼照片影片，或是使用網頁應用程式開發產品，使用網頁來互動，在這過渡期中也產生了很多新的協議規範，才造就了現在的Ｗ eb2.0 時代。（Web2.0，可以讀跟寫，但資料都是儲存在某些巨頭公司手上，個人用戶並不具有完全的資料控制權）

而未來所有的資料歸屬可能將會會發展到都轉移至個人加密錢包上，加密錢包可以儲存各式各樣的資料並且加密，也就成了去中心化，檔案只有所有擁有者能夠讀寫或開放權限，但也有一種可能是產物還是可以交由中心化來保存，而產物上傳至區塊鏈來達到不可仿冒作用，所以要非常小心地妥善保管加密錢包，也要有很相當的網路資安觀念，這樣別人就很難去竊取錢包裡的資料，
（Web3，起初的理念是資料不在是儲存於某些大公司們的伺服器，而是分散儲存在世界各地的節點（電腦）上，只有擁有者有權利去讀取、刪除或修改，但未來會怎麼演化也很難講。）

總之覺得網頁的演化是個非常有趣的過程，也很期待未來的網路世界會變得更豐富且有趣跟多樣化。未來上網時也許不僅僅只是上網，也許在上網的時候同時也可能在賺錢與消費，而不是在進到一堆中心化公司，任由這些巨頭公司餵給你的亂七八糟的資料、文字獄、亂刪除、亂封鎖，但這當中還是要重新立下很多新的規範規則，這些也需要些時間去建立跟改進，所以還會是一段長遠的路程，不然網路世界也可能會變得更加混亂。
